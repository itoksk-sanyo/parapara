# parapara

このリポジトリは静的なウェブページ（index.html）を含むシンプルなプロジェクトです。

## 概要

- ファイル: `index.html`
- 種類: 静的サイト（HTML）
- 教科書の端っこに描く「パラパラマンガ」を作成できるウェブアプリです

ローカルでブラウザに直接読み込むか、簡易的なHTTPサーバーで配信して動作を確認できます。

## 動作確認（ローカル）

1. リポジトリをクローンまたはダウンロードします。
2. プロジェクトのルートにある `index.html` をブラウザで開きます。

もしくは、簡易サーバーで確認する方法（Pythonがインストールされている場合の例）:

```bash
# カレントディレクトリをプロジェクトルートに移動してから:
python -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```
